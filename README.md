# sekigae

Treasure 2016 Sekigae Application

## Usage

#### MySQL (Docker)
```
make docker/build
make docker/start
make docker/stop
```

#### Migration
```
make db/create
make db/migrate
make db/clean
```

#### Server
```
make server
```
